# Desenvolvimento Rápido com Soluções de IA

Curso prático de 2 aulas (3h cada) sobre desenvolvimento assistido por Inteligência Artificial.

## 📋 Sobre o Curso

Este curso ensina como usar ferramentas de IA — especialmente o **GitHub Copilot** — para aumentar a produtividade no desenvolvimento de software. Os exercícios usam **Python** com dados do domínio **bancário**.

## 🗂️ Estrutura

```
├── aula1/              # Fundamentos e Primeiros Passos
│   ├── demos/          # Scripts de demonstração ao vivo
│   ├── labs/           # Exercícios práticos
│   └── dados/          # CSVs de dados bancários
│
├── aula2/              # Produtividade e Projeto Final
│   ├── demos/          # Scripts de demonstração
│   ├── labs/           # Exercícios e mini projeto
│   └── dados/          # CSVs para consolidação
│
└── recursos/           # Links úteis e referências
```

## 🚀 Como Usar

### Pré-requisitos
- Python 3.10+
- Visual Studio Code
- Extensão GitHub Copilot (conta Free)

### Instalar dependências
```bash
pip install -r requirements.txt
```

## 📚 Aulas

### Aula 1 — Fundamentos e Primeiros Passos (3h)
- O que é desenvolvimento assistido por IA
- Fundamentos de LLMs
- Panorama de ferramentas (Copilot, ChatGPT, Cursor, etc.)
- GitHub Copilot na prática
- Python + Copilot: automação bancária

### Aula 2 — Produtividade e Projeto Final (3h)
- IA no ciclo completo de desenvolvimento
- Refatoração e debugging com Copilot
- Pipeline de dados bancários
- Documentação automática
- Mini projeto final
